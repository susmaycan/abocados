# aBocados 🥑

Application to search for recipes made with Vue, Nuxt and Typescript.

## Steps to Setup

1. Install dependencies

```bash
yarn install
```

2. Run in development mode

```bash
yarn dev
```
3. To build in production mode, first create build

```bash
yarn build
```

4. Then type the following command:

```bash
yarn start
```

5. Open the following url on your browser: <http://localhost:3000>
